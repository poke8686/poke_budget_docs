# 스마트 가계부 (Smart Budget)

나의 소비를 자동으로 기록하고, 한눈에 파악하는 똑똑한 가계부 앱

---

## 주요 기능

### 자동 가계부 입력
카드 결제, 계좌 이체 알림을 자동으로 인식하여 가계부에 기록합니다.
일일이 입력할 필요 없이, 결제 알림이 오면 자동으로 기록됩니다.

**지원 앱:**
- 문자 메시지 (카드 승인 SMS)
- 주요 카드사 앱 (신한, 삼성, KB, 현대, 롯데, 하나, 우리 등)
- 은행 앱 (이체 알림)
- 카카오톡 (송금 알림)
- 간편결제 (토스, 카카오페이, 네이버페이)

### 간편 수기 입력
자동 입력이 안 되는 현금 사용도 간편하게 기록할 수 있습니다.
- 최근 사용처 자동완성
- 반복 지출 자동 등록 (월세, 구독료 등)
- 빠른 입력 위젯

### 다양한 통계 & 분석
내 소비 습관을 다양한 각도에서 분석합니다.
- **기간별 통계:** 일별 / 주별 / 월별 / 연별
- **카테고리별 분석:** 어디에 가장 많이 쓰는지 한눈에
- **지출 추이:** 월간 소비 변화 추적
- **예산 관리:** 설정한 예산 대비 지출 현황
- **카드/계좌별:** 어떤 결제수단을 많이 쓰는지
- **사용처 순위:** 자주 가는 곳 TOP 랭킹
- **소비 패턴:** 요일별/시간대별 소비 습관

### 예산 관리
매월 예산을 설정하고 지출을 관리하세요.
- 전체 예산 및 카테고리별 예산 설정
- 예산 초과 시 알림으로 알려줌
- 남은 예산 실시간 확인

### 데이터 내보내기 & 공유
기록한 데이터를 다양한 형식으로 내보내고 공유할 수 있습니다.
- **내보내기 형식:** CSV, Excel, PDF
- **공유:** 이메일, 카카오톡 등으로 바로 전송
- **월간 리포트:** 한 달 소비 요약 리포트 자동 생성

### 백업 & 클라우드 동기화
- **로컬 백업:** 기기 내부 저장소에 단일 파일로 자동 덮어쓰기
- **자동 백업:** 7일마다 백그라운드 자동 저장
- **클라우드 백업:** QR 코드로 다른 기기에 데이터 옮기기 (72시간 유효)
- **딥링크 복원:** 다른 기기에서 링크 열면 앱이 자동 실행되어 복원

### 프리미엄 / 쿠폰
- 쿠폰을 등록하면 광고가 제거됩니다
- 여러 쿠폰을 등록하면 기간이 누적 연장됩니다
- 만료 전까지 모든 광고(배너/전면/리워드) 자동 우회

### 보안
금융 데이터를 안전하게 보호합니다.
- 앱 잠금 (PIN 번호 / 지문 / 얼굴 인식)
- 데이터 암호화 저장 (SQLCipher)

## 사용 방법

### 1. 앱 설치 후 초기 설정
1. 앱을 설치하고 실행합니다.
2. 온보딩 화면에서 기본 설정을 완료합니다.
3. **알림 접근 권한**을 허용합니다. (자동 입력에 필요)
4. 자동 스캔할 앱을 선택합니다. (카드사, 은행 앱 등)

### 2. 자동 입력 설정
1. **설정 > 알림 스캔** 메뉴로 이동합니다.
2. 스캔할 앱 목록에서 원하는 앱을 켜거나 끕니다.
3. 이후 해당 앱에서 결제/이체 알림이 오면 자동으로 기록됩니다.

> **참고:** 앱이 알림을 정확하게 인식하지 못하는 경우, 알림 내용과 함께 오류를 제보해 주시면 빠르게 개선하겠습니다.

### 3. 수기 입력
1. 메인 화면의 **+** 버튼을 탭합니다.
2. 금액, 카테고리, 사용처를 입력합니다.
3. 필요 시 메모를 추가하고 저장합니다.

### 4. 통계 확인
1. 하단 탭에서 **통계** 메뉴를 선택합니다.
2. 기간, 카테고리 등 원하는 필터를 설정합니다.
3. 다양한 차트와 분석을 확인합니다.

### 5. 예산 설정
1. **예산** 메뉴로 이동합니다.
2. 월별 총 예산 또는 카테고리별 예산을 설정합니다.
3. 메인 화면에서 남은 예산을 실시간으로 확인합니다.

### 6. 데이터 내보내기
1. **설정 > 데이터 내보내기** 메뉴로 이동합니다.
2. 내보낼 기간과 형식(CSV/Excel/PDF)을 선택합니다.
3. 이메일, 카카오톡 등 원하는 방법으로 전송합니다.

### 7. 백업 & 복원
1. **설정 > 백업** 메뉴로 이동합니다.
2. **로컬 저장:** "저장" 버튼을 누르면 기기 내부에 백업 파일이 덮어쓰기 저장됩니다.
3. **클라우드 백업:** 기기를 옮길 땐 클라우드 백업으로 QR 코드를 발급받아 다른 기기에서 스캔하세요. (72시간 유효)
4. 기기 변경 시 백업 파일 또는 QR로 데이터를 복원할 수 있습니다.

### 8. 광고 제거 (프리미엄)
1. **설정 > 일반 > 프리미엄 / 쿠폰** 메뉴로 이동합니다.
2. 쿠폰 코드를 입력하고 "쿠폰 등록"을 누릅니다.
3. 등록 즉시 모든 광고가 제거됩니다.

## 앱 권한 안내

| 권한 | 용도 | 필수 여부 |
|-----|------|---------|
| 알림 접근 | 카드/은행 알림 자동 인식 | 선택 (자동 입력 사용 시 필수) |
| 저장소 | 데이터 내보내기, 로컬 백업 | 선택 (해당 기능 사용 시) |
| 인터넷 | 클라우드 백업, 쿠폰 인증, 광고 | 필수 |

> 이 앱은 선택한 앱의 알림만 읽으며, 다른 앱의 알림에는 접근하지 않습니다.
> 알림 원문은 저장하지 않으며, 추출된 거래 정보(금액, 사용처, 날짜)만 기록합니다.
> 클라우드 백업은 사용자가 명시적으로 업로드한 경우에만 서버에 72시간 임시 저장됩니다.

## 자주 묻는 질문 (FAQ)

**Q. 알림을 허용했는데 자동 입력이 안 됩니다.**
A. 설정 > 알림 스캔에서 해당 앱이 활성화되어 있는지 확인해 주세요. 또한 기기의 배터리 최적화 설정에서 이 앱을 예외로 추가해 주세요.

**Q. 잘못 입력된 거래를 수정할 수 있나요?**
A. 네, 거래 목록에서 해당 거래를 탭하면 수정하거나 삭제할 수 있습니다.

**Q. 데이터가 서버에 저장되나요?**
A. 기본적으로 모든 데이터는 사용자의 기기에만 저장됩니다. **클라우드 백업** 기능을 사용해 업로드한 경우에만 서버에 72시간 임시 저장되며, 그 후 자동 삭제됩니다.

**Q. 앱을 삭제하면 데이터가 사라지나요?**
A. 네. 로컬 백업 파일 또는 클라우드 백업(QR)을 만들어 두면 재설치 후 복원할 수 있습니다.

**Q. 광고를 제거할 수 있나요?**
A. 네, **설정 > 일반 > 프리미엄 / 쿠폰**에서 쿠폰을 등록하면 광고가 제거됩니다. 이벤트 쿠폰은 별도 공지를 통해 안내합니다.
